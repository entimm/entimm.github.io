vagrant 常用操作

发表于:2016-04-24，更新于:2016-08-29，By enjoy
大纲
  1. 1. 查看全局状态
  2. 2. 基本安装流程，以centos为例
  3. 3. 销毁
  4. 4. 打包
  5. 5. 状态

查看全局状态

vagrant global-status

基本安装流程，以centos为例

vagrant add centos cent.box

vagrant init centos

vagrant up

vagrant ssh

销毁

vagrant destroy

vagrant box remove centos

打包

vagrant package –output newcentos.box

状态

vagrant halt # 关闭虚拟机
vagrant reload # 重启虚拟机
